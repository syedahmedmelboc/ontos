## Intersection Device Specification

### CSS Utility Classes

Add to `index.css` in `@layer utilities`:

```css
/* Brand Victoria Intersection Device - +/-25.3 degree gradient accent */
.intersection-device {
  background: linear-gradient(
    25.3deg,
    hsl(var(--primary)) 0%,
    hsl(var(--secondary)) 50%,
    hsl(var(--accent)) 100%
  );
}

/* Horizontal bar variant for dividers */
.intersection-device-bar {
  height: 2px;
  width: 100%;
  background: linear-gradient(
    25.3deg,
    hsl(var(--primary)) 0%,
    hsl(var(--secondary)) 50%,
    hsl(var(--accent)) 100%
  );
}

/* Reverse direction variant */
.intersection-device-reverse {
  background: linear-gradient(
    -25.3deg,
    hsl(var(--primary)) 0%,
    hsl(var(--secondary)) 50%,
    hsl(var(--accent)) 100%
  );
}
```

### Application Targets

| Component | Location | Implementation |
|-----------|----------|----------------|
| Header divider | Below sticky header | Add `<div className="intersection-device-bar" />` after header element |
| Sidebar logo divider | Below logo area | Add `<div className="intersection-device-bar" />` after logo container |

### Design Constraints

1. **Angle:** Exactly 25.3deg (or -25.3deg for reverse) - per Brand Victoria guidelines
2. **Height:** 2px for standard dividers, 4px for prominent accents
3. **Color source:** Must use CSS variables (`--primary`, `--secondary`, `--accent`) - never hardcoded hex
4. **Placement:** Reserved for key brand moments only - not for every divider
